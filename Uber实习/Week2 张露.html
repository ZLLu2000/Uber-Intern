Week 2：Uber平台与竞品价格因素分析¶

  网约车平台并不像地铁/巴士等公共交通工具一样拥有恒定价格。通常来说，网约车的行程价格跟里程、时间、需求、气候状况、当地交通便利程度等因素都有联系。
  接下来，我们有一个较为庞大的数据集，时长为一周，囊括了从Uber和Lyft两个平台的行程数据以及同期的气象数据。请根据数据内容和实际的地图地理位置信息尝试分析：

  • 行程长度和价格的关系;
  • 行程价格与气候的关系;
  • 行程类型与价格的关系;

  同时尝试总结，在时域和空域上这些数据的特征。比如，早晨常见的迁移方向是什么？傍晚常见的迁移方向是什么？工作日的行程和周末的行程特征是否有具体区别？等等等等，总结出一个报告，尝试给出针对司机优化运营的统计和方案。

数据预处理¶

首先导入数据，然后将时间戳转换为时间类型数据,并去除包含缺失值的记录。

import pandas as pd
import numpy as np
from time import localtime,strftime
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['axes.unicode_minus']=False #显示负号
plt.rcParams['font.sans-serif']=['SimHei'] #中文显示
data1 = pd.read_csv(r"weather.csv")#气象数据
data2 = pd.read_csv(r"cab_rides.csv")#行程数据
#将两个表格中的时间戳转换成时间数据
def get_date(t):
    if len(str(t))==10:
        times = strftime("%Y-%m-%d %H:%M:%S",localtime(t))
    else:
        times = strftime("%Y-%m-%d %H:%M:%S",localtime(t/1000))
    return times
data1["time_stamp"] = data1["time_stamp"].apply(get_date)
data2["time_stamp"] = data2["time_stamp"].apply(get_date)
data1 = data1.dropna()
data2 = data2.dropna()

from datetime import datetime
data = []
for i in data2.sample(10000).values:#从行程表中随机抽取10000条记录进行分析
    distance = i[0]
    source = i[4]
    price = i[5]
    cab_type = i[1]
    T = i[2]
    times=[]
    d = data1[data1["location"]==source]
    for j in d.values:
        abj = datetime.strptime(T,"%Y-%m-%d %H:%M:%S")-datetime.strptime(j[5],"%Y-%m-%d %H:%M:%S")
        times.append(abs(abj.total_seconds()))
    value = d.values[times.index(min(times))]
    data.append([distance,cab_type,source,price,T,value[0],value[2],value[3],value[4],value[6],value[7]])
data = pd.DataFrame(data,columns=["distance","cab_type","source","price","time_stamp","temp","clouds",

五、总结¶

  经过对行程记录数据的可视化与挖掘分析，发现Uber和Lyft两个平台的行程长度和价格相关系数均处于0.3-0.5区间范围，相关程度属于中等偏弱的水平。同时分别对其价格和行程长度作线性回归分析发现，Uber平台平均起步价为10.45元，行程长度每增加1公里，价格增加2.44元；Lyft平台平均起步价为10.06元，行程长度每增加1公里，价格增加3.33元。结合同时期气候数据发现，各气候因素与价格的相关性系数绝对值均小于0.1，因此，在观察时间段内气候对价格变动的影响程度非常低。在相同的行程长度下，Uber平台的价格异常值数量要低于Lyft平台,价格平均值基本接近。
  行程记录在时域和空域上的特征表现为，上午的人员主要流动方向为South Station到Financial District以及Boston University到West End；傍晚的人员主要流动方向为Beacon Hill到North End以及Financial District到Haymarket Square。工作日的人员主要流动方向为North End到Back Bay以及Financial District到South Station；周末的人员主要流动方向为Financial District到Haymarket Square以及South Station到Financial District。
